# DecodeLabs Responsive Web Layout

## Description

This project is a fully responsive web layout created as part of the DecodeLabs Project 2 assignment. The website demonstrates modern responsive web design techniques using pure HTML, CSS, and JavaScript without any external frameworks.

The project focuses on building adaptive layouts that work smoothly across desktop, tablet, and mobile devices.

---

## Features

* Fully responsive design
* Sticky navigation bar
* Mobile hamburger menu
* CSS Grid and Flexbox layouts
* Fluid typography using clamp()
* Responsive cards and sections
* Interactive contact form
* Dark mode support
* Accessibility-friendly design
* Mobile-first approach

---

## Technologies Used

* HTML5
